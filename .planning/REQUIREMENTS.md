# Requirements: Studency

**Defined:** 2026-04-06
**Core Value:** Единая карточка клиента через всю воронку — от первого касания до оплаты и обучения

## v1 Requirements

### Авторизация

- [ ] **AUTH-01**: Пользователь может зарегистрироваться (email + пароль)
- [ ] **AUTH-02**: Пользователь может войти в систему
- [ ] **AUTH-03**: Пользователь может сбросить пароль
- [ ] **AUTH-04**: Сессия сохраняется между обновлениями страницы

### Проекты

- [ ] **PROJ-01**: Пользователь может создать проект
- [ ] **PROJ-02**: Пользователь может привязать домен к проекту
- [ ] **PROJ-03**: Внутри проекта доступны все модули (воронки, CRM, боты, сайты, обучение, аналитика)
- [ ] **PROJ-04**: Пользователь может переключаться между проектами

### Воронки

- [ ] **FUNL-01**: Пользователь может создать воронку с названием
- [ ] **FUNL-02**: Пользователь может добавлять этапы в воронку
- [ ] **FUNL-03**: На каждом этапе можно подключить инструмент (бот, лендинг, обучение)
- [ ] **FUNL-04**: Воронка визуализируется как последовательность этапов
- [ ] **FUNL-05**: Аналитика считает конверсию между этапами воронки

### CRM

- [ ] **CRM-01**: Автоматическое создание карточки клиента при первом касании
- [ ] **CRM-02**: Карточка содержит: контакты, историю действий, заказы, чат, прогресс обучения
- [ ] **CRM-03**: CRM-доска с колонками по этапам воронки
- [ ] **CRM-04**: Возможность кликнуть на клиента из любого модуля и открыть его карточку
- [ ] **CRM-05**: Возможность вручную добавлять данные в карточку
- [ ] **CRM-06**: Уникальные ссылки переносят данные клиента между платформами

### Чат-боты

- [ ] **BOT-01**: Создание чат-бота через AI (описание → бот)
- [ ] **BOT-02**: Бот работает в Telegram
- [ ] **BOT-03**: Список всех клиентов бота с этапами
- [ ] **BOT-04**: Просмотр полного чата с каждым клиентом
- [ ] **BOT-05**: Настройка дожимов на каждом этапе
- [ ] **BOT-06**: Рассылки по всей базе или сегменту
- [ ] **BOT-07**: Структура воронки бота визуализируется

### Лендинги

- [ ] **LAND-01**: Генерация лендинга через AI (описание → страница)
- [ ] **LAND-02**: Лендинг хостится на домене проекта
- [ ] **LAND-03**: Кнопка оплаты на лендинге → Продамус
- [ ] **LAND-04**: Трекинг посещений → данные в карточку клиента

### Учебная платформа

- [ ] **EDU-01**: Создание продукта (курса) с уроками
- [ ] **EDU-02**: Урок = видео + текст/материалы
- [ ] **EDU-03**: Доступ к продукту открывается после оплаты
- [ ] **EDU-04**: Отслеживание прогресса (какие уроки прошёл клиент)

### Заказы и оплата

- [ ] **PAY-01**: Создание заказа (привязка клиент + продукт)
- [ ] **PAY-02**: Статусы заказа: создан, оплачен, отменён
- [ ] **PAY-03**: Интеграция с Продамус (создание ссылки на оплату)
- [ ] **PAY-04**: Вебхук от Продамус → автоматическая смена статуса заказа
- [ ] **PAY-05**: После оплаты → автоматическое открытие доступа к продукту

### Аналитика

- [ ] **ANLT-01**: Дашборд по воронке: каждый этап + конверсии
- [ ] **ANLT-02**: Отдельная аналитика по чат-боту (без воронки)
- [ ] **ANLT-03**: Отдельная аналитика по лендингу (без воронки)
- [ ] **ANLT-04**: Отдельная аналитика по учебной платформе (без воронки)
- [ ] **ANLT-05**: Из аналитики можно кликнуть на клиента → открыть карточку CRM

## v2 Requirements

### Масштабирование

- **SCALE-01**: Мультитенантность (несколько пользователей платформы)
- **SCALE-02**: Биллинг и тарифные планы
- **SCALE-03**: Реферальная программа

### Дополнительные интеграции

- **INTG-01**: WhatsApp Business API
- **INTG-02**: Instagram интеграция (уникальные ссылки)
- **INTG-03**: Дополнительные платёжные системы (ЮKassa, Tinkoff)
- **INTG-04**: Email-рассылки

### Расширенная аналитика

- **ANLT-06**: A/B тестирование лендингов
- **ANLT-07**: Когортный анализ
- **ANLT-08**: LTV клиента

## Out of Scope

| Feature | Reason |
|---------|--------|
| Мобильное приложение | Сложный SaaS, с телефона не используют |
| Drag-and-drop конструктор сайтов | AI генерирует — проще и быстрее |
| Визуальный конструктор ботов (блоки) | AI-агент вместо этого |
| Множественные платёжные системы | Только Продамус на v1 |
| WhatsApp / Instagram | Только Telegram на v1 |
| Email-маркетинг | Фокус на мессенджерах |
