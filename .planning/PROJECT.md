# Studency

## What This Is

Единая маркетинговая платформа для создания воронок продаж. Объединяет CRM, чат-боты, лендинги, учебную платформу и аналитику в одном сервисе с единой карточкой клиента. Создана маркетологом для маркетологов — замена связки GetCourse + Salebot + Tilda.

## Core Value

Единая карточка клиента, которая автоматически собирает все действия через всю воронку — от первого касания в Telegram до оплаты и прохождения курса. Ни один существующий сервис этого не делает.

### Out of Scope

- Мобильная версия — сложный сервис, с телефона не пользуются
- Визуальный конструктор сайтов (drag-and-drop) — AI генерирует вместо этого
- Визуальный конструктор чат-ботов (блоки) — AI-агент вместо этого
- Множественные платёжные системы — только Продамус на v1
- WhatsApp интеграция — только Telegram на v1
- Мультитенантность / биллинг / рефералка — v2, сначала для себя

## Context

- Основатель — маркетолог с 5+ годами опыта, 40-50 потенциальных клиентов
- Текущая боль: GetCourse, Salebot, Tilda не интегрированы, аналитику собирать по частям, техспец стоит 20-40к
- Типичная воронка простая: 2-3 видео → лендинг → кнопка оплаты → дожимы в чат-боте
- Первая версия для личного использования, затем продукт для клиентов
- Опыт с Supabase есть (проект MyFinanceAI)
- Дизайн-референс: GoHighLevel — минимализм, боковая навигация

## Constraints

- **Stack**: Next.js + Supabase + Vercel + Claude API + Telegram Bot API + Продамус
- **Design**: Минимализм, белый + #6A55F8 градиент, только desktop
- **Platform**: Только web (desktop), мобильная версия не нужна
- **AI**: Claude API для генерации чат-ботов и лендингов
- **Payments**: Только Продамус на v1
- **Messenger**: Только Telegram на v1
